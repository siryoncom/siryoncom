# utils.vehicle.generateRandomPlate

```lua
utils.vehicle.generateRandomPlate()
```
This function generate a random plate in a vehicle

### Dependency
```lua
local utils = M("utils")
```

### Example
```lua
local randomPlate = utils.vehicle.generateRandomPlate()

print(randomPlate) -- Returns a random plate
```
